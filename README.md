# RAG-AI-Production-app
Production-ready Retrieval-Augmented Generation (RAG) application built with FastAPI, Inngest, OpenAI, and Streamlit. Features event-driven PDF ingestion, semantic search, vector embeddings, durable workflow execution, and context-aware AI responses.

## Features 

PDF Ingestion Pipeline -
Upload and process documents asynchronously
Automatic text extraction and chunking
Retrieval-Augmented Generation (RAG)
Semantic search over embedded documents
Context-aware responses using retrieved chunks

Vector Search -
Embeddings generated using OpenAI
Similarity search for relevant document retrieval
Event-Driven Architecture
Built with Inngest for durable background workflows
Handles ingestion, processing, and indexing reliably

FastAPI Backend -
High-performance API for ingestion and querying
Modular and production-ready structure

Streamlit Frontend -
Simple UI for uploading documents and querying the system
Real-time AI responses with retrieved context

Scalable Workflow Design -
Decoupled ingestion and query pipelines
Designed for production deployment

## Tech Stack
Backend: FastAPI

Frontend: Streamlit

AI / LLM: OpenAI API

Workflow Orchestration: Inngest

Embeddings / RAG: Vector search

Language: Python 3.10+
